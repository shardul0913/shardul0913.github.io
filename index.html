Transfer Learning Application on Kaggle Dataset: Tacking Stars with exoplanets

The dataset for the Exo planet tracking had light flux features for each star. The Tensor of 5087*3198 was time based light flux reading. I took the approach of applying Convlutional Neural Nets to understand the pattern changes. I applied Parameter Transfer Learning for VGG16, I learnt the technique of working with 2 Channel datasets in 3 Channel NNs and the freezing Convolutional layers of pre-trained network.

Movie Genre Prediction :Building a robust multi label classifier in Apache Spark

Traditional Sklearn libraries support multi label classificatio but in PySpark Mllib has few limitations for multi lable classification. By one hot encoding the genres and leveraging parallel processing of Sprak RDDs, I designed an Logistic Regression model for binary classification of all genres while testing out different feature engineering techniques to keep time complexity low

Football game predition using tree based models. A Collaboration with Dublin Business School students

Implemented XG Boosting and Random Forest Bagging Approach to Football prediction problem statement with Grid search Hyperparameter tuning for better accuracy.

Fraud Detection tool to track anomalies in Travel Expenses data.

Algorithm uses One Class SVM to track anomalies on Multi Variate Time Series. The Travel Expense Predition is calculated using L2 Regression

A Twitter sentiment analyzing software

Analysis of tweets based on the sentimentality and polarity with Twitter API using Python, BeautifulSoup, textblob and Tkinter

Tracking anomalous trend in Machine Sensors to trigger a maintainance call

Based on the trends in the normal machine sensor data I trained an Autoencoder LSTM to detect any out of pattern sensor data. I used ARIMA model to track the individual machine sensor data trend for better fault proof approach

Ensighten Marketing Tag Management API

Examined Effect of ‘US China Trade War’ on Domestic and International Companies
